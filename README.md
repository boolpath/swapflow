# Swapflow

## Setup
Install:
- brew install erlang elixir
- brew install postgresql
- mix local.hex
- mix archive.install hex phx_new

Create project:
- mix phx.new swapflow
- mix ecto.create

Start Phoenix:
- mix setup
- mix phx.server

Run in production:
- [Deployment guides](https://hexdocs.pm/phoenix/deployment.html).
